# 40Study Web

Next.js frontend cho nền tảng 40Study.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Structure

```
src/
├── app/           # App router pages
├── components/    # React components
├── hooks/         # Custom hooks
├── lib/           # Utilities
├── services/      # API services
└── types/         # TypeScript types
```

## Getting Started

```bash
npm install
npm run dev
```
